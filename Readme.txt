# Food Delivery Web App

This is a food delivery web app that allows users to order food from various restaurants and have it delivered to their doorstep. The app provides a seamless user experience and offers a wide range of food options to choose from.

## Features

- User registration and login
- Restaurant listing and search
- Menu browsing and item selection
- Cart management
- Order placement and tracking
- Payment integration
- Delivery status updates

## Technologies Used

- Front-end: HTML, CSS, JavaScript
- Back-end: Node.js, Express.js
- Database: MongoDB
- Payment Integration: Stripe

## Installation

1. Clone the repository: `git clone https://github.com/your-username/food-delivery-web-app.git`
2. Install the dependencies: `npm install`
3. Set up the database connection in `config.js`
4. Start the server: `npm start`

## Usage

1. Open the web app in your browser.
2. Register a new account or log in with your existing credentials.
3. Browse through the list of restaurants and select your desired items.
4. Add the items to your cart and proceed to checkout.
5. Enter your delivery address and complete the payment.
6. Track the status of your order until it is delivered.
